# Setup a Linux Machine in CloudVeneto for Compiling and Running C Codes

This guide explains how to create a Linux virtual machine (VM) in CloudVeneto and use it to compile and run simple C programs.

## 1. Log in to CloudVeneto
Open the CloudVeneto dashboard in a browser and sign in with your account. After login, choose the correct project if needed.

## 2. Add an SSH key
Before creating the VM, prepare an SSH key.

- Go to **Compute → Key Pairs**
- Create a new key pair or import an existing public key
- Save the private key safely

If needed, fix the permissions of the private key:

```bash
chmod 600 mykey.pem
```

## 3. Create the Linux VM
Launch a new instance from **Compute → Instances**.

Recommended settings:
- **Name:** any clear name, for example `c-vm`
- **Image:** **AlmaLinux** or **Ubuntu**
- **Flavor:** a small flavor is enough
- **Network:** your project network, usually something like `<ProjectName>-lan`
- **Key Pair:** select the SSH key you prepared

Click **Launch Instance** and wait until the status becomes **Active**.

## 4. Connect to the VM with SSH
CloudVeneto VMs are often on a private network. In that case, connect through the gate machine.

Example:

```bash
ssh -J <your_username>@gate.cloudveneto.it -i mykey.pem almalinux@<VM_PRIVATE_IP>
```

If the VM is directly reachable, you can connect with:

```bash
ssh -i mykey.pem almalinux@<VM_IP>
```

Typical usernames are:
- `almalinux` for AlmaLinux
- `ubuntu` for Ubuntu
- `centos` for CentOS

## 5. Install the C compiler
Once inside the VM, install GCC.

For AlmaLinux:

```bash
sudo dnf install gcc -y
```

For Ubuntu:

```bash
sudo apt update
sudo apt install gcc -y
```

Check the installation:

```bash
gcc --version
```

## 6. Create a simple C program
Create a file called `hello.c`:

```bash
nano hello.c
```

Paste this code:

```c
#include <stdio.h>

int main() {
    char name[50];

    printf("Enter your name: ");
    scanf("%s", name);

    printf("Hello %s, welcome to C programming!\n", name);

    return 0;
}
```

Save and exit.

## 7. Compile the program
Compile the source file with GCC:

```bash
gcc -Wall -Wextra -o hello hello.c
```

Here:
- `hello.c` is the source file
- `hello` is the executable file
- `-Wall -Wextra` enables useful warnings

## 8. Run the program
Run the compiled file with:

```bash
./hello
```


## 9. Conclusion
With CloudVeneto, we can create a Linux VM, access it using SSH, install GCC, and compile and run C programs. This gives a simple and useful environment for practicing C programming.
